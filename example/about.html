About this Template

More information will be coming soon once I have finalized most of the template and the various portions of the template. For now it includes the basic framework, which can be shrinked using modern CSS3 media queries, 6 types of btns in 3 different sizes, headings that respond to the size to the viewable screen size. Stay tuned for more features added daily.
